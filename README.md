# MedicosExpress

Este proyecto es una plataforma médica de alto rendimiento diseñada para clínicas en Reynosa, Tamaulipas.

## 🚀 Características

- **Astro:** Framework de alto rendimiento.
- **Tailwind CSS:** Estilos modernos y responsivos.
- **React:** Interactividad en el formulario de citas.
- **Automatización:** Integración con n8n para gestión de citas.

## 🧞 Comandos

| Comando | Acción |
| :--- | :--- |
| `npm install` | Instala las dependencias |
| `npm run dev` | Inicia el servidor de desarrollo en `localhost:4321` |
| `npm run build` | Compila el sitio para producción |
| `npm run preview` | Previsualiza la compilación localmente |

## 📁 Estructura del Proyecto

- `src/components/`: Componentes de Astro y React.
- `src/layouts/`: Diseños compartidos.
- `src/pages/`: Rutas del sitio.
- `src/config/`: Configuración dinámica de clientes (Médicos/Clínicas).
- `context/`: Documentación y planes de desarrollo.
